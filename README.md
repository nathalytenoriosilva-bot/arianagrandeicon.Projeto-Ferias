# Projeto-Ferias - Ariana Grande Fanpage 

## Objetivo
Criar uma página inicial sobre a cantora Ariana Grande.

## Tecnologias Usadas
- HTML
- CSS
- Bootstrap 
- JavaScript

## Itens do JS Criados
1. Alerta inicial de boas-vindas.
2. Botão de contraste (Modo Escuro).
3. Botão funcional de voltar ao topo.
4. Validação de segurança no formulário de contato.
5. Inserção automática da data atual no rodapé.
